## 2. Positioning Encoding 

Before Transformer, RNN reads an input sequentially, which is very slow. Transformer architecture changed this by putting all the words for LLM to process in parallel. But this brings a new issue: the computer can't tell the order of the words. Transformer forced a 'positioning ruler' into the digital vector of each word. For instance, it assigns 1 for "I" and 2 for "like".
